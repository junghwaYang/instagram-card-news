# Gemini 3.1 Pro 리서치

## 출시 정보
- 출시일: 2026년 2월 19일
- 개발사: Google DeepMind
- 이전 모델: Gemini 3 Pro

## 핵심 벤치마크 성과
- **ARC-AGI-2**: 77.1% (Gemini 3 Pro 31.1% → 2배 이상 향상)
- **SWE-Bench Verified**: 80.6% (에이전틱 코딩)
- **GPQA Diamond**: 94.3% (과학 지식)
- **MMLU-Pro**: 업계 최고 수준

## 주요 사양
- 입력 컨텍스트: 1M 토큰 (100만 토큰)
- 출력 한도: 65,536 토큰 (65k)
- Deep Think 모드 도입 (복잡한 추론에 특화)
- 애니메이션 SVG 생성 기능

## 접근 채널
- Google AI Studio
- Gemini CLI
- Vertex AI
- Gemini 앱
- NotebookLM

## 핵심 변화 포인트
1. 추론 능력 2배 이상 향상 (ARC-AGI-2 기준)
2. 에이전틱 코딩 능력 대폭 강화
3. Deep Think 모드로 복잡한 문제 해결
4. 100만 토큰 컨텍스트로 대규모 문서 처리
5. 과학/수학 분야 최고 수준 달성
